Course Enroller

This application allows students to submit their course preferences for the upcoming semester. The system will then allocate courses based on availability and student preferences.

Current number of preferences submitted: {{ preference_count }}

Student Use Cases

  1. Submit Your Course Preferences
    To submit your course preferences, click on the "Enter Preferences" link below. You will be taken to a form where you can enter your student information and rank your preferred courses.
    Enter Preferences
  2. View Registrations
    After the allocation has been performed, students can view their allocated courses by clicking on the "View Registrations" link.
    View Registrations
  3. View Course List
    Admins and students can view the list of courses and their current availability status.
    View Course List

Admin Use Cases

  1. Perform Allocation
    Once all students have submitted their preferences, an admin can run the allocation algorithm to assign students to courses based on their preferences and course availability.
    Perform Allocation
  2. Generate Sample Preferences
    Admins can generate sample preferences to test the allocation algorithm. This will create a set of sample student preferences and run the allocation.
    Generate Sample Preferences
  3. Clear Database
    Admins can clear the entire database to reset the application for a new semester or testing.
    Clear Database
